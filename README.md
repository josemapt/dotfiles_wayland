# dotfiles_wayland
Personal config of 100% wayland arch install

# software
- sway (WM)
- alacritty (terminal)
- mako (notifications)
- brave (browser)
- visual-studio-code-wayland (editor)
- waybar (statusbar)
- fish (shell)
- wofi (application launcher)
- grim (screenshot)
- gammastep (redshift for wayland)
- swayidle and swaylock-effects (screen locking)
